## ğŸ”§ ADVANCED PANEL (Click "ADVANCED")

### Filter Envelope
- **F-ATTACK** - How fast filter opens (1ms - 5s)
- **F-DECAY** - How fast filter decays (1ms - 5s)
- **F-SUSTAIN** - Filter level held (0-100%)
- **F-RELEASE** - How fast filter closes (1ms - 5s)
- **F-AMOUNT** - Envelope depth (0-100%)

### Amp Envelope
- **A-ATTACK** - How fast sound fades in (1ms - 5s)
- **A-DECAY** - How fast sound decays (1ms - 5s)
- **A-SUSTAIN** - Volume level held (0-100%)
- **A-RELEASE** - How fast sound fades out (1ms - 5s)

### LFO (Wobble)
- **LFO RATE** - Wobble speed (0.01 - 20 Hz)
- **LFO AMOUNT** - Wobble depth (0-100%)

### Velocity
- **VELâ†’FILT** - How velocity affects brightness (0-100%)
- **VELâ†’AMP** - How velocity affects volume (0-100%)

### Misc
- **KEY TRACK** - Higher notes brighter (0-100%)
- **VOICES** - Unison voices (1-5)
- **SUB OCT** - Sub octave (-1 or -2 octaves)

## ğŸ› TROUBLESHOOTING

**No Sound?**
- Turn up SUB MIX
- Turn up CUTOFF to 2-5kHz
- Check MIDI is playing

**Too Quiet?**
- Add more SUB MIX
- Add DRIVE
- Increase track volume

**Distorted?**
- Lower DRIVE
- Lower RESONANCE
- Turn down track volume

**Won't Load?**
- Restart DAW
- Rescan plugins
- Check correct folder
- macOS: Allow in Security settings
